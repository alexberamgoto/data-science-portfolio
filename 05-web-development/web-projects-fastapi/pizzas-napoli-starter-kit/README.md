# Pizzas Napoli Starter Kit

Projet modèle pour le développement d'une API REST avec Bun, TypeScript et Hono (à compléter).

## Variables d'environnement

- Créer un fichier .env à la racine du projet basé sur le fichier .env.example fourni.

## Commandes utiles

- Installation des dépendances NPM :

```sh
bun install
```

- Démarrage

```shx
bun start
```

- Démarrage en mode Hot Reload

```sh
bun run dev
```

open <http://localhost:3000>
