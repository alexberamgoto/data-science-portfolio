# Elysia + Bun — Mini API TypeScript

API TypeScript ultra-légère basée sur **Elysia** et le runtime **Bun** — exploration de l'écosystème JS moderne sans Node.js.

## Stack
- [Bun](https://bun.sh) (runtime + bundler + package manager)
- [Elysia](https://elysiajs.com/) (framework HTTP)
- TypeScript

## Lancer
```bash
bun install
bun run index.ts
```

> Initialement créé via `bun init` (Bun v1.3.8).
