# Power BI — Tableau de bord COVID-19 par classe d'âge (France)

Reporting Power BI sur les données hospitalières COVID-19 ventilées par tranche d'âge (DREES, France).

## Contenu
| Fichier | Rôle |
|---------|------|
| `TP1 powerbi.pbix` | Premier rapport — distribution âge × hospitalisations |
| `tttt.pbix` | Itération avancée du tableau de bord |
| `donnees-hospitalieres-classe-age-covid19-2021-02-27-19h03.csv` | Source : `data.gouv.fr` |
| `2026-TP1/` | TP guidé |
| `202602-TP2/` | TP avancé (mesures DAX, slicers) |

## Compétences mises en œuvre
- Modélisation Power BI (relations, hiérarchies)
- DAX (mesures calculées, filtres temporels)
- Visualisations (cartes, slicers, KPI)
- Sources publiques data.gouv.fr

## Lancer
Ouvrir les fichiers `.pbix` avec **Power BI Desktop**.
