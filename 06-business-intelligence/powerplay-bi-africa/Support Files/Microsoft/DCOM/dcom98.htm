Install Microsoft's Distributed COM 1.3 for Windows 98

To install DCOM 1.3 for Windows 98 you must download it from the Microsoft Web site. To do this,

  • Go to http://www.microsoft.com.
  • Click Downloads.
  • Search using the keyword search.
  • Type DCOM98 as the keyword.
  • Select Windows 98 as the operating system.
  • Select 'All Downloads' from the 'Show Results for' drop down list.
  • Click on 'Find it' button.
