# IBM Cognos PowerPlay — BI Afrique (Indicateurs socio-économiques)

Projet de Business Intelligence sous **IBM Cognos PowerPlay** : conception d'un cube OLAP et d'un tableau d'indicateurs socio-économiques africains (santé, éducation, démographie).

## Livrables
| Fichier | Description |
|---------|-------------|
| `Powerplay.ats` | Définition du cube OLAP (Application Transformer) |
| `Projet BI.txt` | Notes techniques sur la modélisation |
| `Collection des indicateurs pour le Projet BI/` | Indicateurs sources |
| `collectionpourleprojetbi/` | Données structurées |
| `TP BI/` | Travaux pratiques BI Afrique (CSV/XLSX d'indicateurs) |
| `Support Files/` | Fichiers de support du cube |
| `WhatsApp Image *.jpeg` | Captures d'écran du tableau de bord final |

## Compétences
- Modélisation OLAP multidimensionnelle (dimensions, mesures, hiérarchies)
- Conception d'un **datawarehouse** thématique
- IBM Cognos suite (Transformer, PowerPlay)
- Sources de données ouvertes africaines

> Le binaire IBM Cognos n'est pas inclus (logiciel propriétaire). Pour visualiser le cube, installer Cognos PowerPlay puis ouvrir `Powerplay.ats`.
