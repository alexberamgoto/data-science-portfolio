# Deep Learning — TPs CNN & Génération musicale

Travaux pratiques de Deep Learning : réseaux convolutifs (vision), apprentissage supervisé, et génération musicale à partir de fichiers MIDI.

## Notebooks

| Notebook | Sujet |
|----------|-------|
| `DeepLearning.ipynb` | Cours appliqué — perceptron, MLP, backprop |
| `Td_deepLearning.ipynb` | TD — réseaux de neurones from scratch |
| `TP_apprentissage_deep_learning.ipynb` | Apprentissage supervisé profond |
| `TP_projet_d'IA_cnn.ipynb` | **Projet CNN** — classification d'images |

## Données
Fichiers `.mid` (suites de Bach pour violoncelle) — utilisés pour expérimenter la **génération musicale par RNN/LSTM**.

## Stack
Python 3.11 · TensorFlow / Keras · PyTorch · numpy · matplotlib · music21
