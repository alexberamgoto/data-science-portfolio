# Machine Learning — TPs & Études de cas

Notebooks d'apprentissage couvrant les principales familles d'algos ML supervisés et non supervisés.

## Contenu

| Notebook | Sujet | Techniques |
|----------|-------|------------|
| `customer-segmentation-with-rfm-analysis.ipynb` | Segmentation client | Analyse RFM, K-Means |
| `StopWords.ipynb` | NLP — Stop words & prétraitement texte | tokenisation, TF-IDF |
| `flashcards_ml.py` | Générateur PDF de flashcards ML | FPDF, synthèse pédagogique |
| `TD1/` | Classification supervisée | scikit-learn |
| `facebook+live+sellers+in+thailand/` | Clustering | UCI dataset, K-Means |

## Stack
Python 3.11 · pandas · scikit-learn · matplotlib · seaborn · FPDF
